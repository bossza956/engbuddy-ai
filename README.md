# 🌟 ยินดีต้อนรับสู่ EngBuddy AI! 🌟

**EngBuddy AI** คือเว็บแอปพลิเคชันส่วนตัวสำหรับการฝึกฝนภาษาอังกฤษ ออกแบบมาเพื่อเน้นเทคนิค **"เรียนรู้ผ่านเรื่องเล่าสั้นๆ และการฝึกจำประโยคนำไปใช้จริง (Sentence Mining & Input-Based Learning)"** โดยมีสมองกลอัจฉริยะอย่าง **Google Gemini 2.5 Flash (Gemini Pro)** คอยช่วยเหลือในการแปล วิเคราะห์ประโยค และแต่งเรื่องราวน่าอ่านให้กับคุณ

โปรเจกต์นี้เขียนขึ้นด้วย HTML, CSS และ JavaScript ธรรมดา **คุณไม่จำเป็นต้องลงโปรแกรมเขียนโค้ดหรือเขียนโปรแกรมใดๆ ทั้งสิ้น** เพียงแค่ทำตามขั้นตอนด้านล่างนี้ ก็สามารถใช้งานบนคอมพิวเตอร์ของคุณได้ทันที!

## 🚀 วิธีเริ่มต้นใช้งานใน 1 นาที

### ขั้นตอนที่ 1: ดึงไฟล์โปรเจกต์มาเปิดใช้งาน
1. เข้าไปที่โฟลเดอร์ `Eng Personal Practice` ในคอมพิวเตอร์ของคุณ
2. ค้นหาไฟล์ที่ชื่อว่า **`index.html`**
3. **ดับเบิลคลิก (Double-click)** ที่ไฟล์ `index.html` เพื่อเปิดขึ้นมาบนเบราว์เซอร์ของคุณ (เช่น Google Chrome, Microsoft Edge, Safari)
   *(คุณจะเห็นหน้าจอเว็บแอปแสนสวยปรากฏขึ้นมาทันที!)*

---

### ขั้นตอนที่ 2: วิธีสมัครขอรับ API Key ของ Gemini ฟรี!
เพื่อให้เว็บแอปสามารถแต่งเรื่องและแปลภาษาได้ คุณต้องมี API Key ของตนเอง ซึ่งสมัครได้ **ฟรี 100%** ไม่มีค่าใช้จ่าย:

1. เปิดเว็บเบราว์เซอร์และเข้าไปที่หน้าเว็บ [Google AI Studio (https://aistudio.google.com/)](https://aistudio.google.com/)
2. ล็อกอินด้วยบัญชี **Google (Gmail)** ของคุณ
3. มองหาปุ่มคำว่า **"Get API key"** (ส่วนใหญ่อยู่มุมซ้ายบน)
4. คลิก **"Create API key"** จากนั้นเลือกสร้างคีย์ใหม่
5. **คัดลอก (Copy)** รหัส API Key ที่ขึ้นต้นด้วย `AIzaSy...` เก็บไว้
6. กลับมาที่เว็บแอป EngBuddy AI ของเรา -> ไปที่แท็บ **"ตั้งค่าระบบ (Settings)"** (เมนูล่างสุดซ้ายมือ)
7. วางรหัสลงในช่อง **Gemini Pro API Key** จากนั้นกดปุ่ม **"บันทึกคีย์ (Save)"**
8. *(แนะนำ)* กดปุ่ม **"ทดสอบการเชื่อมต่อ (Test)"** เพื่อเช็คว่าคีย์ใช้งานได้ถูกต้องหรือไม่

## 📖 วิธีฝึกภาษาอังกฤษให้เก่งด้วย EngBuddy AI

แอปพลิเคชันนี้ออกแบบมาเพื่อรองรับสไตล์การเรียนรู้ที่คุณชอบ (อ่านเรื่องสั้นซ้ำๆ ให้จำประโยคได้และนำไปใช้งาน):

### 1. โหมดสร้างเรื่องเล่าและแยกประโยค (Stories)
- เลือก **ระดับภาษา** ที่เหมาะกับคุณ (ง่าย ปานกลาง หรือยาก)
- เลือกหัวข้อด่วนที่สนใจหรือพิมพ์ภาษาไทยบอกหัวเรื่องสั้นๆ (เช่น "สั่งเบอร์เกอร์", "แนะนำตัว")
- กด **"สร้างเรื่องเล่าด้วย AI"**
- **วิธีฝึก**: 
  1. ค่อยๆ อ่านเรื่องเล่าทีละประโยค
  2. กด **ไอคอนลำโพง 🔊** เพื่อฟังเสียงการออกเสียงที่ถูกต้อง ฝึกออกเสียงตาม (Shadowing) ซ้ำๆ 3-5 ครั้ง
  3. กดดูความหมายภาษาไทยและศัพท์สำคัญใต้ประโยค
  4. หากชอบประโยคไหนเป็นพิเศษ ให้กด **ไอคอนริบบิ้น 🔖 (Save)** เพื่อเก็บประโยคนั้นมาฝึกในหน้า Flashcard

### 2. โหมดทบทวนการ์ดประโยคสะสม (Cards)
- ประโยคที่คุณกดบันทึก (Save) ทั้งหมดจะถูกเก็บไว้ที่นี่แบบถาวรในคอมพิวเตอร์ของคุณ
- **วิธีฝึก**: 
  1. การ์ดจะแสดงประโยคภาษาอังกฤษ ให้คุณกดฟังเสียงและฝึกอ่านออกเสียง
  2. คลิกที่การ์ดเพื่อ **พลิกการ์ด (Flip)** ดูคำแปลภาษาไทยและโน้ตคำศัพท์
  3. หากคุณท่องจำและนำประโยคนี้ไปใช้ได้คล่องแคล่วแล้ว ให้กดปุ่ม **"จำประโยคนี้ได้แล้ว (Learned)"** เพื่อจัดเก็บประโยคนี้เข้าแฟ้มเรียนรู้เสร็จสิ้น

### 3. โหมดเล่นควิซทบทวนความจำ (Quiz)
- ระบบจะดึงประโยคที่คุณบันทึกสะสมไว้มาสร้างแบบทดสอบให้ฝึกทบทวน
- **โหมดเติมคำในช่องว่าง (Fill-in-the-Blank)**: ระบบจะปิดคำศัพท์สำคัญไว้ ให้คุณฝึกสะกดและพิมพ์ศัพท์ตัวนั้นลงในประโยค
- **โหมดเรียงประโยค (Sentence Reorder)**: ระบบจะสลับตำแหน่งคำศัพท์ทั้งหมด ให้คุณคลิกเรียงประโยคภาษาอังกฤษกลับมาให้ถูกต้อง
- กดปุ่ม **"Check"** เพื่อตรวจสอบความถูกต้อง หากทำถูก Gemini จะออกเสียงประโยคนั้นให้คุณฟังอีกครั้ง

### 4. โหมดแชทกับคุณครู AI (Chat Tutor)
- ฝึกสวมบทบาทพูดคุยสนทนาจริงกับ AI ติวเตอร์ โดยมีติวเตอร์ 3 สไตล์:
  - **Friendly Friend (เพื่อนคุยชิลๆ)**: ชวนคุยทั่วไป มีจุดผิดจะแก้ศัพท์ให้ดูเป็นธรรมชาติอธิบายเป็นภาษาไทยสั้นๆ ท้ายบทสนทนา
  - **Strict Grammar Coach (ครูแกรมม่าสุดเนี๊ยบ)**: สแกนตรวจแกรมม่า ละเอียดเป็นข้อๆ และเสนอวิธีเขียนที่ดีกว่า
  - **Business Mentor (โค้ชวัยทำงาน)**: แนะนำวิธีใช้วลีภาษาอังกฤษให้ดูเป็นมืออาชีพ สุภาพ และเป็นทางการ

### 5. คลังคำศัพท์สะสม (Vocab)
- ทุกประโยคที่คุณกดเซฟ ระบบจะแยกคำศัพท์เด่นๆ มาบันทึกเข้าพจนานุกรมส่วนตัวของคุณตรงนี้อัตโนมัติ คุณสามารถกดค้นหาศัพท์ ลบคำศัพท์ หรือกดฟังเสียงอ่านศัพท์รายตัวได้ตลอดเวลา
